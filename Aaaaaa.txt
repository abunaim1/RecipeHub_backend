To show the SQL code generated by Django using MySQL Workbench, you'll need to make sure that your Django project is configured to use a MySQL database. Here’s how you can achieve this:

### Step-by-Step Guide

1. **Install MySQL Client for Python**
   Make sure you have the MySQL client library installed in your virtual environment:

   ```bash
   pip install mysqlclient
   ```

2. **Configure Django to Use MySQL**
   Update your `settings.py` to use MySQL:

   ```python
   # settings.py
   DATABASES = {
       'default': {
           'ENGINE': 'django.db.backends.mysql',
           'NAME': 'your_database_name',
           'USER': 'your_database_user',
           'PASSWORD': 'your_database_password',
           'HOST': 'localhost',  # or your database server IP
           'PORT': '3306',
       }
   }
   ```

3. **Define Your Model**
   Make sure your `Student` model is defined in `myapp/models.py`:

   ```python
   # myapp/models.py
   from django.db import models

   class Student(models.Model):
       name = models.CharField(max_length=100)
       age = models.IntegerField()
       enrollment_date = models.DateField()

       def __str__(self):
           return self.name
   ```

4. **Create and Apply Migrations**
   Run the following commands to create and apply migrations:

   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

5. **Generate SQL Code**
   Use the `sqlmigrate` command to see the SQL for the migration:

   ```bash
   python manage.py sqlmigrate myapp 0001
   ```

### Example Output in MySQL Syntax
After running the `sqlmigrate` command, you will get SQL output like this (adjusted for MySQL syntax):

```sql
BEGIN;
--
-- Create model Student
--
CREATE TABLE `myapp_student` (
    `id` integer AUTO_INCREMENT NOT NULL PRIMARY KEY,
    `name` varchar(100) NOT NULL,
    `age` integer NOT NULL,
    `enrollment_date` date NOT NULL
) ENGINE=InnoDB;
COMMIT;
```

### Import SQL into MySQL Workbench

1. **Open MySQL Workbench**
   Open MySQL Workbench and connect to your MySQL server.

2. **Create a New Query Tab**
   Click on the "New Query Tab" button.

3. **Paste the SQL Code**
   Copy the SQL code generated by Django and paste it into the query tab.

4. **Execute the SQL Code**
   Click the lightning bolt icon to execute the SQL code. This will create the `myapp_student` table in your MySQL database.

### Detailed Output in MySQL Workbench

Once you run the SQL commands in MySQL Workbench, the output will look something like this:

```sql
-- Connecting to MySQL server and switching to the database
USE your_database_name;

-- Begin the transaction
BEGIN;

-- Create the Student table
CREATE TABLE `myapp_student` (
    `id` integer AUTO_INCREMENT NOT NULL PRIMARY KEY,
    `name` varchar(100) NOT NULL,
    `age` integer NOT NULL,
    `enrollment_date` date NOT NULL
) ENGINE=InnoDB;

-- Commit the transaction
COMMIT;
```

By following these steps, you can show your SQL code and the actual creation of the table in MySQL Workbench, fulfilling the requirements of your DBMS project.
